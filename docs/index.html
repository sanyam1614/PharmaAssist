Features

Everything you need to understand your medication

Built with Perplexity's Sonar API, PharmaAssist retrieves accurate pharmaceutical data with full source attribution.

🔍

Real-time database search

Queries live medical databases on every request, ensuring your information is always current and accurate.

📚

Cited sources

Every response includes references from authoritative organizations like FDA, NIH, MedlinePlus, and WHO.

🔊

Text-to-speech output

Listen to medication information hands-free. Audio responses are generated automatically for each query.

💬

Multi-turn conversations

Ask follow-up questions naturally. PharmaAssist maintains context throughout your entire conversation.

💾

Download history

Export your full conversation as a text file to refer back to medication information whenever you need.

⚡

Streamlit powered

Clean, fast web interface built with Streamlit — easy to run locally or deploy to the cloud in minutes.

How it works

From question to answer in seconds

A streamlined three-step process that brings verified pharmaceutical knowledge to your fingertips.

1

Ask your question

Type the name of any medication or pharmaceutical query into the chat interface.

2

AI searches databases

Perplexity's Sonar API queries real-time medical databases across trusted health sources.

3

Get cited answers

Receive a comprehensive, sourced response with optional audio playback and download.
